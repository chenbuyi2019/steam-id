Steam ID 转换

本工具适用于转换Steam个人用户和Steam组的ID ，仅是数学上的转换，不能把昵称变成数字 ID 。

从玩家链接获取ID64 ：在链接的末尾加上 ?xml=1 ，比如 我的
从Steam组获取ID64 ：在链接的末尾加上 /memberslistxml/?xml=1 ，比如 Crowbar 组




在下方输入要转换的 ID ，一行一个，工具会自动提取行内的数字。

各种 Steam ID

名字 样本 说明
id64 76561198099466387
103582791434761767 		最常用的id表达形式
但是太长，数字太大，excel 和 js 的默认数字类型没法处理
76开头的是玩家，文本长度是17
10开头的是群组，文本长度是18
id32 STEAM_0:1:69600329 STEAM_0 和 STEAM_1 都是id32，并且是同一个意思。
只是部分场合用_0，部分场合用_1。
一般 Source 游戏内都是用_0
第一个:之后的整段才是关键信息
id3 [U:1:139200659]
[g:1:5240359] 		id3 也就是好友代码、群组代码
[U:1:打头就是用户了， [g:1: 打头就是群组。
第二个:之后的数字才是关键信息
好友代码本质上是 id32 的第三个数字乘以2的积加上第二个数字
玩家的完整 id3 可以直接接在 https://steamcommunity.com/profiles/ 之后作为超链接访问
